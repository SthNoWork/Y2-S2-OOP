public class Main {
  public static boolean isPrime(int n, int i) {
    if (n <= 2)    return (n == 2);
    if (n % i == 0) return false;
    if (i * i > n)  return true;
    return isPrime(n, i + 1); // recursive
  }

  public static void main(String[] args) {
    Scanner input = new Scanner(System.in);
    System.out.print("Input a number: ");
    int num = input.nextInt();
    if (isPrime(num, 2)) {

    }
  }
}

Vowel / Consonant Counter
char iteration · toLowerCase · counters
Source
public class Main {
  public static void main(String[] args) {
    Scanner input = new Scanner(System.in);
    System.out.print("Input a sentence: ");
    String sentence = input.nextLine().toLowerCase();

    int vowel = 0, consonant = 0, space = 0;

    for (int i = 0; i < sentence.length(); i++) {
      char ch = sentence.charAt(i);
      if ("aeiou".indexOf(ch) >= 0) vowel++;
      else if (ch >= 'a' && ch <= 'z') consonant++;
      else if (ch == ' ') space++;
    }
    System.out.println("Vowels: "    + vowel);
    System.out.println("Consonants: " + consonant);
    System.out.println("Spaces: "    + space);
  }
}
Output
Input a sentence: Hello World
Count of vowel is: 3
Count of consonant is: 7
Count of space is: 1

Biggest Number in Dynamic Array
ArrayList · Collections.max · while loop
Source
import java.util.ArrayList;
import java.util.Collections;

public class Main {
  public static void main(String[] args) {
    Scanner input = new Scanner(System.in);
    ArrayList<Integer> arr = new ArrayList<>();
    char choice = 'y';
    int index = 0;

    while (choice == 'y' || choice == 'Y') {
      System.out.print("Value at index " + index + ": ");
      arr.add(input.nextInt());
      System.out.print("Add more (y/n)?: ");
      choice = input.next().charAt(0);
      index++;
    }
    System.out.println("Biggest: " + Collections.max(arr));
  }
}
Output
Value at index 0: 42
Add more (y/n)?: y
Value at index 1: 99
Add more (y/n)?: y
Value at index 2: 7
Add more (y/n)?: n
The biggest number among in Array is: 99

        case 1: System.out.println(A + " + " + B + " = " + (A+B)); break;
        case 2: System.out.println(A + " - " + B + " = " + (A-B)); break;
        case 3: System.out.println(A + " x " + B + " = " + (A*B)); break;
        case 4: if (B==0) System.out.println("Cannot divide by zero.");

      }
    }
  }
}
